#  Product Card

A modern **Product Card UI** built using **HTML5** and **CSS3** as part of my CSS learning journey. This project focuses on creating a clean product card layout while practicing Flexbox, image styling, gradients, hover effects, and button design.

## Preview

> Add a screenshot of the project here after uploading it to your repository.

##  Features

* Clean and modern product card design
* Large product image with thumbnail gallery
* Product name and category
* Current and original price display
* Interactive hover effects on product thumbnails
* Stylish "Add to Cart" button
* Gradient background section
* Centered layout using Flexbox
* Smooth hover animations

## 🛠️ Built With

* HTML5
* CSS3
* Flexbox

##  What I Learned

* Creating card-based layouts
* Centering elements using Flexbox
* Styling images with `object-fit`
* Using CSS transitions and transforms
* Applying linear gradients
* Working with spacing, padding, and margins
* Designing buttons with hover effects
* Organizing HTML and CSS for reusable components
